# Frequency tune panel

The panel is intended to be used as part of analog radio stack.  
It is designed to be used with Arduino-based control software (like Mobiflight).

## Used hardware  
Custom PCB with:  
1. three 4-pin XH2.54 connectors:   
    * COM frequency dual rotary encoder
    * NAV frequency dual rotary encoder
    * Swap buttons and Rotary buttons
2. four 6-digit 7-segment displays with 16-pin dual-raw male headers
    * the header is intended to be used with proprietary TM1637-based addon board

## Features:  
* two double rotary encoders for frequency tuning - Mhz and KHz 
* two frequency swap buttons

## Supplied configuration files
* Mobiflight - XPlane 11+
* Mobiflight - MSFS 2020+
