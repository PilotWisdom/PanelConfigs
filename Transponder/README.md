### Nano board pinout (D13 LED needs to be taken off)  

 pins **D2-D9** buttons  0-7  
 pins **D10-D13,A0,A1** - buttons OFF,SBY,ON,ALT,VFR,ID  
 pins **A2,A3** - CLK,DIN @Squak code (TM1637)  
 pins **A4,A5** - CLK,DIN @Mode (TM1637)
