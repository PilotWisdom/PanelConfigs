# PilotWisdom configuration repository
PilotWisdom panels and modules configuration files.
Please visit the PilotWisdomShop on Etsy: 
https://pilotwisdomshop.etsy.com

## Folder structure
Each folder represent indiviual panel/module.
Folder content depends on the item type.

### Mobiflight-based panels
The folder typically includes:
- Used hardware description and pinout
- The underlying controller's configuration for MobiFlight
- Mobiflight configuration file(s) for X-Plane 11/12
- Mobiflight configuration file(s) for Microsoft Flight Simulator 2020/2024
- README file with purchase links

### Joystick-based panels
The folder typically includes:
- Used hardware description and pinout
- Suggested configuration screenshots/description for X-Plane 11/12
- Suggested configuration screenshots/description for Microsoft Flight Simulator 2020/2024
- README file with purchase links and firmware link (if applicable)
